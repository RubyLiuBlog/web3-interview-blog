12:Td6d,<h3><strong>JavaScript 数组去重方法总结</strong></h3>
<h4><strong>1. 使用 <code>Set</code>（ES6 最简单方式）</strong></h4>
<pre><code class="language-javascript">const arr = [1, 2, 2, 3, 4, 4];
const uniqueArr = [...new Set(arr)]; // [1, 2, 3, 4]
</code></pre>
<ul>
<li><strong>优点</strong>：代码简洁，性能较好。</li>
<li><strong>缺点</strong>：无法处理对象引用去重（<code>Set</code> 认为 <code>{} !== {}</code>）。</li>
</ul>
<h4><strong>2. <code>filter</code> + <code>indexOf</code>（ES5 兼容）</strong></h4>
<pre><code class="language-javascript">const arr = [1, 2, 2, 3, 4, 4];
const uniqueArr = arr.filter((item, index) => arr.indexOf(item) === index);
</code></pre>
<ul>
<li><strong>优点</strong>：兼容旧浏览器。</li>
<li><strong>缺点</strong>：时间复杂度 O(n²)，性能较差（大数组不推荐）。</li>
</ul>
<h4><strong>3. <code>reduce</code> 去重</strong></h4>
<pre><code class="language-javascript">const arr = [1, 2, 2, 3, 4, 4];
const uniqueArr = arr.reduce(
  (acc, cur) => (acc.includes(cur) ? acc : [...acc, cur]),
  []
);
</code></pre>
<ul>
<li><strong>优点</strong>：灵活，可结合复杂逻辑。</li>
<li><strong>缺点</strong>：每次迭代创建新数组（性能一般）。</li>
</ul>
<h4><strong>4. 对象键值去重（适用于基本类型）</strong></h4>
<pre><code class="language-javascript">const arr = [1, 2, 2, 3, 4, 4];
const uniqueArr = Object.keys(
  arr.reduce((acc, cur) => ((acc[cur] = true), acc), {})
).map(Number); // 注意：键会被转为字符串
</code></pre>
<ul>
<li><strong>优点</strong>：一次遍历，性能较好。</li>
<li><strong>缺点</strong>：仅适用于可转为字符串的值（如数字、字符串）。</li>
</ul>
<h4><strong>5. 对象数组去重（按属性）</strong></h4>
<pre><code class="language-javascript">const arr = [{ id: 1 }, { id: 2 }, { id: 1 }];
const uniqueArr = arr.filter(
  (item, index, self) => index === self.findIndex((obj) => obj.id === item.id)
);
</code></pre>
<ul>
<li><strong>适用场景</strong>：根据对象某个属性（如 <code>id</code>）去重。</li>
<li><strong>缺点</strong>：<code>findIndex</code> 嵌套导致性能较差（大数组慎用）。</li>
</ul>
<h4><strong>6. <code>Map</code> 去重（支持对象引用）</strong></h4>
<pre><code class="language-javascript">const arr = [{ id: 1 }, { id: 2 }, { id: 1 }];
const map = new Map();
arr.forEach((item) => map.set(item.id, item)); // 按 id 去重
const uniqueArr = [...map.values()];
</code></pre>
<ul>
<li><strong>优点</strong>：高性能，支持复杂对象。</li>
</ul>
<hr>
<h3><strong>性能对比（大数组优先选 <code>Set</code> 或 <code>Map</code>）</strong></h3>
<p>| 方法               | 时间复杂度 | 适用场景           |
| ------------------ | ---------- | ------------------ |
| <code>Set</code>              | O(n)       | 基本类型去重       |
| <code>Map</code>              | O(n)       | 对象按属性去重     |
| <code>filter</code>+<code>indexOf</code> | O(n²)      | 小数组或兼容性场景 |</p>
<p><strong>推荐选择</strong>：</p>
<ul>
<li><strong>简单数组</strong> → <code>[...new Set(arr)]</code></li>
<li><strong>对象数组</strong> → <code>Map</code> 或 <code>reduce</code> + 属性判断</li>
<li><strong>兼容旧环境</strong> → <code>filter</code> + <code>indexOf</code></li>
</ul>
