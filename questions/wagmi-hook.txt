12:Tc6d,<h1>Wagmi Hooks 使用经验</h1>
<h2>核心连接 Hooks</h2>
<ol>
<li>
<p><strong><code>useConnect</code></strong></p>
<ul>
<li>连接钱包</li>
<li>获取可用连接器列表</li>
<li>处理连接状态和错误</li>
</ul>
</li>
<li>
<p><strong><code>useDisconnect</code></strong></p>
<ul>
<li>断开钱包连接</li>
<li>清理会话状态</li>
</ul>
</li>
<li>
<p><strong><code>useAccount</code></strong></p>
<ul>
<li>获取当前连接账户信息</li>
<li>监听账户变化</li>
<li>检查连接状态</li>
</ul>
</li>
<li>
<p><strong><code>useNetwork</code></strong></p>
<ul>
<li>获取当前网络信息</li>
<li>监测网络变化</li>
<li>检查支持的网络</li>
</ul>
</li>
<li>
<p><strong><code>useSwitchNetwork</code></strong></p>
<ul>
<li>切换网络功能</li>
<li>处理网络切换错误</li>
</ul>
</li>
</ol>
<h2>合约交互 Hooks</h2>
<ol>
<li>
<p><strong><code>useContractRead</code></strong></p>
<ul>
<li>读取合约状态</li>
<li>自动重新获取数据</li>
</ul>
</li>
<li>
<p><strong><code>useContractWrite</code></strong></p>
<ul>
<li>执行合约写入操作</li>
<li>处理交易状态</li>
</ul>
</li>
<li>
<p><strong><code>usePrepareContractWrite</code></strong></p>
<ul>
<li>预先验证合约写入操作</li>
<li>获取 gas 估算</li>
</ul>
</li>
<li>
<p><strong><code>useContractEvent</code></strong></p>
<ul>
<li>监听合约事件</li>
<li>处理事件回调</li>
</ul>
</li>
</ol>
<h2>交易 Hooks</h2>
<ol>
<li>
<p><strong><code>useTransaction</code></strong></p>
<ul>
<li>获取交易详情</li>
</ul>
</li>
<li>
<p><strong><code>useSendTransaction</code></strong></p>
<ul>
<li>发送原生代币交易</li>
</ul>
</li>
<li>
<p><strong><code>usePrepareSendTransaction</code></strong></p>
<ul>
<li>预先验证交易参数</li>
</ul>
</li>
<li>
<p><strong><code>useWaitForTransaction</code></strong></p>
<ul>
<li>等待交易完成</li>
<li>处理交易确认</li>
</ul>
</li>
</ol>
<h2>签名 Hooks</h2>
<ol>
<li>
<p><strong><code>useSignMessage</code></strong></p>
<ul>
<li>签名消息</li>
<li>处理签名结果</li>
</ul>
</li>
<li>
<p><strong><code>useSignTypedData</code></strong></p>
<ul>
<li>签名 EIP-712 类型化数据</li>
</ul>
</li>
</ol>
<h2>余额和代币 Hooks</h2>
<ol>
<li>
<p><strong><code>useBalance</code></strong></p>
<ul>
<li>获取原生代币余额</li>
<li>自动刷新余额</li>
</ul>
</li>
<li>
<p><strong><code>useToken</code></strong></p>
<ul>
<li>获取 ERC20 代币信息</li>
</ul>
</li>
<li>
<p><strong><code>useTokenBalance</code></strong></p>
<ul>
<li>获取 ERC20 代币余额</li>
</ul>
</li>
</ol>
<h2>其他实用 Hooks</h2>
<ol>
<li>
<p><strong><code>useEnsName</code>/<code>useEnsAddress</code></strong></p>
<ul>
<li>ENS 名称解析</li>
</ul>
</li>
<li>
<p><strong><code>useBlockNumber</code></strong></p>
<ul>
<li>获取当前区块高度</li>
</ul>
</li>
<li>
<p><strong><code>useFeeData</code></strong></p>
<ul>
<li>获取当前 gas 费用信息</li>
</ul>
</li>
<li>
<p><strong><code>useChainId</code></strong></p>
<ul>
<li>获取当前链 ID</li>
</ul>
</li>
</ol>
<p>注：我的使用经验基于 Wagmi v1.x 版本，具体 hook 可能随着版本更新有所变化。</p>
